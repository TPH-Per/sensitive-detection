# Stack

## Web app
- React 19 + TypeScript
- Vite
- React Router 7
- Zustand for state
- Tailwind CSS + styled-components
- React Hook Form + Zod
- Firebase (Auth, Firestore, RTDB, Storage, FCM)
- ZegoCloud for audio/video calling
- Lucide React, Emoji Picker, React Easy Crop, date-fns

## Backend
- Firebase Cloud Functions v2
- Admin SDK + Realtime Database orchestration
- Callable HTTPS functions and event-driven handlers

## Mobile app
- Flutter + Dart 3.3+
- Provider state management
- Firebase client SDKs
- image/file/media packages, Google Maps, Zego UI kits
